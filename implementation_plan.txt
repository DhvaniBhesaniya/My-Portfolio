# Dhvani's Portfolio ‚Äî React SPA Conversion Plan

## Overview

Convert the existing vanilla HTML/CSS/JS portfolio at `/home/dhvani/personal_work/My-Portfolio` into a modern React single-page application. The existing site uses `index.html` + `style.css` + `main.js` in the root with assets already in `public/assets/`.

---

## Key Decisions

> [!IMPORTANT]
> **Scaffold inside the existing repo root.** The existing `public/assets/` directory (images, resume, certificates) already exists and is correctly structured. We will scaffold the Vite app **directly in the project root**, NOT in a subfolder, so the existing assets are preserved in `public/`.

> [!WARNING]
> The prompt forbids TypeScript. Every generated file must be `.js` or `.jsx`. No `tsconfig.json`, no `.ts`/`.tsx`. Shadcn init must be configured to use JavaScript.

> [!NOTE]
> Magic UI components must be **manually written** (copy-paste source) ‚Äî they are not installable via npm. We'll write idiomatic JSX versions of `Particles`, `TypingAnimation`, and `MagicCard`.

---

## Tech Stack

| Layer | Tool |
|---|---|
| Build | Vite (latest) ‚Äî `vite.config.js` |
| Framework | React 19, `.jsx` files only |
| Styling | Tailwind CSS v4 (`@tailwindcss/vite`) |
| Components | shadcn/ui (Radix-based) |
| Animation | `motion` package ‚Äî `import from "motion/react"` |
| Animated UI | Magic UI ‚Äî `Particles`, `TypingAnimation`, `MagicCard` (hand-written JSX) |
| Smooth scroll | `lenis` (latest) |
| Icons | `lucide-react` |
| Email | `@emailjs/browser` |
| Routing | None ‚Äî single page, anchor-based scroll |
